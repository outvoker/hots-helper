## 名词解释

**统计量**

- **胜率（WR）**：胜场 ÷ 总局数。
- **WLB（Wilson 置信下界）**：在 95% 置信度下真实胜率的下限。比原始胜率更适合排序。5/5 看似 100%，但 WLB 只 56%；70/100 是 71% WR、WLB 60%，后者更可信。
- **偏移 / lift**：当前胜率与基线的差，单位 **百分点（pp）**。+30pp ≠ "提升 30%"，是绝对差 30 个百分点。
- **p 值**：双比例 z 检验的双尾 p 值。p<0.05 强证据，p<0.10 弱证据，p>0.10 样本不够。

**对局数据**

- **K/D/A**：击杀 / 死亡 / 助攻（K = takedowns − assists，是真实击杀，不是大暴雪原始的 takedowns）。
- **英伤（HeroDamage）**：对敌方英雄造成的总伤害。
- **承伤（DamageTaken）**：受到的总伤害。坦克高、奶位中、刺客低。
- **推塔伤害（StructureDamage）**：对建筑（塔/堡垒/要塞/王座）的伤害。
- **推线伤害（SiegeDamage）**：包括建筑 + 雇佣兵 + 兵线兵。
- **治疗 / 自治疗**：对队友的治疗 / 自己回血。
- **XP / 经验贡献**：清线、击杀、雇佣兵带来的全队经验。全队 100k/场是健康水平，单人 < 5k = 清线不力或挂机。

**团战与控制**

- **团战伤害 / 团战治疗 / 团战承伤**：仅算 5v5 团战瞬间的产出，过滤掉清线刷分。**团战伤害高 + 总英伤低** = 真团队 player；**反过来** = 刷分仔。
- **控时（CC）**：对敌方英雄的控制时长（眩晕 + 缓速 + 沉默 + 定身），单位秒。控制位 / 工具人英雄看这一项。
- **最高连杀 / 多杀**：连续不死的最大击杀数 / 双杀以上次数。爆发收割英雄关注。
- **救人（Escapes）**：替队友抗过致命伤害的次数。
- **复仇击杀（Vengeance）**：在被击杀后 5 秒内反向击杀对方。

**地图目标**

- **雇佣兵营**：占领数（每张图通常 4-8 个）。
- **守望塔**：仅在「咒术高地」「暮光树林」两图。
- **奖项**：每局 11 类 MVP，由游戏自动评判（MVP / 推线 MVP / 治疗 MVP 等）。

仅统计风暴联赛对局；天命乱斗（ARAM）无 BP 已剔除。
